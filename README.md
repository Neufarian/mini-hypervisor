## Features
- Single VM execution in 64-bit mode
- Multiple VM support using POSIX threads
- File operations via virtual IO ports
- Command-line configurable memory and page size
- Safe file handling for shared guest files

## Tech Stack
- **Languages:** C, C++
- **Virtualization:** Kernel-based Virtual Machine (KVM)
- **Concurrency:** POSIX Threads (pthreads)
- **Tools:** GCC, GDB, Make
